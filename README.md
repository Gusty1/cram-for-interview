# 面試抱佛腳

## 緣起

不知道你是不是跟我一樣面試前都會提早到，然後在附近的 7-11 買杯飲料，然後拿起手機拼命的複習面試題目。
而這時某些面試題目的網站可能會有廣告，也有可能你昨天找到的題目今天突然找不到了，所以本 APP 就是為了應付這種情況而生的。
~~用一般的記事本 APP 好像也可以做到一樣的效果...~~
